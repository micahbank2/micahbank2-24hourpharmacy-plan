Phase 0: Global infrastructure (do once)

Legal and business setup
Not started
Checklist
Get an EIN from IRS.gov (free, 5 minutes)
Needed for affiliate signups, bank account, tax filing. Don't give your SSN to every network.
Open a separate business bank account
Any free checking account (Chase, Novo, Mercury, credit union). ALL income in, ALL expenses out. Never mix with personal.
Set up bookkeeping in Wave Accounting (free)
Track every dollar. Date, amount, source, category. You need this for taxes.
Set aside 25-30% of all revenue for taxes
Self-employment tax (15.3%) + income tax. Pay quarterly estimates (Apr 15, Jun 15, Sep 15, Jan 15).

Do soon (months 1-3)

Form an LLC in your state ($50-500, file yourself at Secretary of State website)
Skip LegalZoom. Gives you liability protection on a health-adjacent site.
Get a registered agent or business address if you don't want home address public ($50-150/yr)

Do later (when revenue justifies it)

Hire a CPA ($300-800/yr, when earning $1K+/mo consistently)
General liability insurance ($300-500/yr)
Trademark "24HourPharmacy.com" at USPTO ($250-350, Phase 3+)
S-Corp election (when earning $50K+/yr, saves self-employment tax)
Business credit card (after LLC has revenue history)
What you do NOT need: No pharmacy license, no DEA registration, no NABP accreditation, no HIPAA compliance. You're an information site, not a pharmacy.

How the pieces fit together

Where you build things

This project chat: Planning, strategy, architecture decisions, debugging. Get prompts to paste into Claude Code.

Claude Code: Writes all custom code: theme, plugin, React widgets, Python scripts. Code lives in GitHub repo.

WordPress admin: Content management after code is deployed. Write pages, add pharmacies, enter affiliate codes, manage plugins.

How you deploy

Theme/plugin: Upload zip via WordPress admin or SFTP through Hostinger file manager.

React widgets: Run npm run build locally. Compiled JS outputs into plugin folder. Upload plugin and widget goes with it.

City pages at scale: Run Python data pipeline locally. It pushes pages to WordPress via REST API as drafts. You review and publish in admin.

What changes look like after launch

Content changes (editing pages, publishing posts): WordPress admin. No code.

Affiliate code changes (new BIN/PCN, new programs): WordPress > 24hr Pharmacy > Settings. No code.

Template/style changes: Claude Code, then re-upload theme folder.

Widget updates: Claude Code, run build, re-upload plugin folder.
